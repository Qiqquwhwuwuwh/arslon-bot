# 🤖 Nemis Tili Kursi — Telegram Bot

## Fayllar
- `bot.py` — asosiy bot kodi
- `requirements.txt` — kutubxonalar
- `Procfile` — Railway uchun
- `railway.toml` — Railway sozlamalari

## Railway ga Deploy qilish

1. GitHub da yangi repository oching
2. Shu 4 ta faylni yuklang
3. Railway.app da "New Project" → "Deploy from GitHub"
4. Repository tanlang
5. **Variables** bo'limiga o'ting, qo'shing:
   - `BOT_TOKEN` = sizning bot tokeningiz
   - `GROQ_API_KEY` = sizning Groq kalitingiz
6. Deploy tugmasini bosing — tayyor!
